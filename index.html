Sobre

A Nucleus é uma empresa de tecnologia focada em solucionar problema de forma inteligente para pessoas e empresas.
Trabalhamos com soluções de software, aplicações web, aplicativos mobile, consultoria e aplicações embarcadas.

Aplicações de Software

Trabalhando com as melhores tecnologias do mercado, a Nucleus pode, certamente, suprir qualquer problema para você e sua empresa.
Se você precisa de uma solução para sua empresa, como um banco de dados automatizado por exemplo, conte conosco. A Nucleus irá melhorar o sistema da sua empresa, ou se preferir, fazemos do zero.

Aplicações Mobile

O mundo hoje está conectado. Estar sempre conectado vai lhe dar um passo à frente dos demais.

Aplicações Web

As aplicações web são as comuns no mundo da tecnologia. Toda empresa, seja grande ou pequena, precisa de um site hoje para poder estar no mercado.
Além de sites, podemos fazer completos webservices para solucionar o seu problema.

Sistemas Embarcados

Para resolver problemas de extrema importância, onde as falhas são totalmente inaceitáveis, trabalhamos com sistemas embarcados. Um sistema embarcado consiste em um hardware, um único software e que esses dois componentes trabalhem para fazer apenas uma só coisa.
